Урок А1-24: Повелительное наклонение (Imperativ)

Учимся вежливо просить, давать советы и приказывать.

Lernziele (Цели обучения)

  • Умею вежливо просить (Machen Sie bitte...).
  • Могу давать дружеские советы (Lern fleißig!).
  • Знаю формы Imperativ для du, ihr, Sie и для глаголов с отделяемыми приставками.

1. Что такое Imperativ?

Imperativ — это повелительное наклонение, которое используется для просьб, советов, приказов и указаний. В немецком есть три формы Imperativ:

  • du — для одного близкого человека (друг, ребёнок)
  • ihr — для нескольких близких людей
  • Sie — для вежливого обращения (официально или к незнакомым)

2. Формы Imperativ

ЛицоФормаПримерПеревод
duглагол без окончания + eKomm hier!Иди сюда!
ihrглагол в форме ihr (без -t)Kommt hier!Идите сюда!
Sieглагол + SieKommen Sie hier!Идите сюда! (вежливо)
Запомни!
Для du: глагол без окончания + e (kommen → komm)
Для ihr: обычная форма ihr без -t (kommen → kommt)
Для Sie: инфинитив + Sie (Kommen Sie)

Исключения (без -e): geh, sieh, bleib, lauf, nimm, gib, sieh

3. Вежливые формы (bitte, mal)

Чтобы сделать Imperativ вежливее, добавь bitte (пожалуйста) или mal (давай/ну):

  • Kommen Sie bitte herein. — Войдите, пожалуйста.
  • Gib mir mal das Buch. — Дай-ка мне книгу.
  • Hör mal zu! — Послушай!

4. Глаголы с отделяемыми приставками

У глаголов с отделяемыми приставкой (auf-, ein-, mit-, vor-, zu-) приставка уходит в конец:

  • Mach die Tür auf! — Открой дверь!
  • Steh bitte auf! — Встаньте, пожалуйста!
  • Komm mit! — Иди с нами!
  • Setz dich hin! — Садись!

5. Примеры Imperativ

  • Komm hier! — Иди сюда!
  • Geh nach Hause! — Иди домой!
  • Hör mir zu! — Слушай меня!
  • Nimm das Buch! — Возьми книгу!
  • Sei still! — Будь тихим!
  • Ess dein Gemüse! — Ешь овощи!
  • Trink Wasser! — Пей воду!
  • Schreib eine E-Mail! — Напиши письмо!
