Урок A2-10: Глаголы с управлением — Часть 1

Учимся использовать глаголы с предлогами и правильными падежами.

Lernziele (Цели обучения)

  • Знаю глаголы строго с их предлогами и падежами: warten auf + A, denken an + A, sich freuen auf/über + A.
  • Могу составить с ними правильные примеры.
  • Понимаю, почему важен правильный падеж после глагола.

1. Что такое глаголы с управлением?

Глаголы с управлением (Verben mit Präposition) — это глаголы, которые требуют определённый предлог с определённым падежом. Их нужно учить вместе с предлогом!

Важно: Нельзя сказать "Ich warte die Bus" — нужно "Ich warte auf den Bus" (Akkusativ)!

2. Глагол warten (ждать)

warten auf + Akkusativ — ждать чего-то/кого-то

Глагол + предлогПадежПримерПеревод
warten auf + AkkusativAkkusativIch warte auf den Bus.Я жду автобус.
warten auf + AkkusativAkkusativIch warte auf dich.Я жду тебя.
warten auf + AkkusativAkkusativWir warten auf das Ergebnis.Мы ждём результат.

3. Глагол denken (думать)

denken an + Akkusativ — думать о чём-то/ком-то

Глагол + предлогПадежПримерПеревод
denken an + AkkusativAkkusativIch denke an dich.Я думаю о тебе.
denken an + AkkusativAkkusativIch denke an die Prüfung.Я думаю об экзамене.
denken an + AkkusativAkkusativEr denkt an sein Haus.Он думает о своём доме.

4. Глагол sich freuen (радоваться)

sich freuen auf + Akkusativ — радоваться чему-то (будущее)
sich freuen über + Akkusativ — радоваться чему-то (прошедшее/настоящее)

Глагол + предлогПадежПримерПеревод
sich freuen auf + AkkusativAkkusativIch freue mich auf das Wochenende.Я радуюсь выходным.
sich freuen auf + AkkusativAkkusativIch freue mich auf dich.Я радуюсь тебе (жду с нетерпением).
sich freuen über + AkkusativAkkusativIch freue mich über das Geschenk.Я радуюсь подарку.
sich freuen über + AkkusativAkkusativWir freuen uns über die Nachricht.Мы рады новости.

5. Разница между auf и über

  • sich freuen auf — радоваться будущему событию (Wochenende, Urlaub, Geburtstag)
  • sich freuen über — радоваться прошёлому/настоящему событию (Geschenk, Nachricht, Ergebnis)

6. Примеры всех трёх глаголов

  • Ich warte auf den Bus. — Я жду автобус.
  • Ich denke an dich. — Я думаю о тебе.
  • Ich freue mich auf das Wochenende. — Я радуюсь выходным.
  • Ich freue mich über das Geschenk. — Я радуюсь подарку.
