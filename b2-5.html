Урок B2-5: Пассивный залог с модальными глаголами

Das Problem muss gelöst werden — пассив с müssen, können, sollen, dürfen.

Lernziele (Цели обучения)

  • Строю пассивные предложения с модальными глаголами.
  • Использую конструкции: muss + werden + Partizip II.
  • Понимаю разницу между активом и пассивом с модальными глаголами.

1. Структура пассива с модальными глаголами

В пассиве с модальными глаголами используется конструкция:

СтруктураПримерПеревод
Modalverb + werden + Partizip IIDas Problem muss gelöst werden.Проблема должна быть решена.
Modalverb + Partizip II + werdenDas Problem muss gelöst werden.Проблема должна быть решена.
Важно!
В пассиве с модальными глаголами Infinitiv I (werden) стоит в конце предложения.
Das Buch muss gelesen werden. — Книгу нужно прочитать.
Die Tür kann geöffnet werden. — Дверь может быть открыта.

2. Модальные глаголы в пассиве

Все модальные глаголы могут использоваться в пассиве:

Модальный глаголПримерПеревод
müssenDas Essen muss gekocht werden.Еду нужно приготовить.
könnenDie Tür kann geöffnet werden.Дверь может быть открыта.
sollenDas Auto soll repariert werden.Машину нужно починить.
dürfenDer Brief darf geschrieben werden.Письмо может быть написано.
wollenDas Haus will gebaut werden.Дом хотят построить.
mögenDas Buch mag gelesen werden.Книгу, возможно, читают.

3. Примеры использования

  • Das Problem muss gelöst werden. — Проблема должна быть решена.
  • Die Arbeit kann gemacht werden. — Работа может быть сделана.
  • Das Buch soll gelesen werden. — Книгу нужно прочитать.
  • Der Brief darf geschrieben werden. — Письмо может быть написано.
  • Das Essen muss gekocht werden. — Еду нужно приготовить.
